Step 1
Download the latest Barrier package from:
https://github.com/debauchee/barrier/releases
Select the .deb file for your system architecture (likely amd64).
Step 2
Open a terminal window.
Step 3
Navigate to the directory where the downloaded .deb file is located. You can use the `cd` command to change directories.
Step 4
Install the Barrier package using the following command:
sudo dpkg -i barrier-*.deb
Step 5
If there are any missing dependencies, you can resolve them by running:
sudo apt-get install -f
Step 6
After the installation is complete, you can start Barrier by searching for it in your application menu or by running:
barrier in the terminal.
Step 7
On the Barrier main screen, select whether this machine will act as a server (controlling other machines) or a client (being controlled by another machine).
Step 8
If you selected server, configure the screen positions by dragging the monitor icons to match the physical layout of your machines.
Step 9
If you selected client, enter the server's IP address in the connection field.
Step 10
Click "Start" to initiate the connection. The client should connect to the server and you should be able to control the client machine using the server's keyboard and mouse.
Step 11
To ensure Barrier starts automatically on boot, you can add it to your startup applications. This process may vary depending on your desktop environment.
Step 12
For GNOME, you can use the "Startup Applications" tool to add Barrier.
Step 13
For XFCE, you can add a new entry in "Session and Startup" under the "Application Autostart" tab.
Step 14
For other desktop environments, refer to their respective documentation for adding startup applications.
Step 15
Ensure that your firewall settings allow Barrier traffic. Barrier uses TCP port 24800 by default. You may need to create a rule to allow traffic on this port.
Step 16
To add a firewall rule, use the following command:
sudo ufw allow 24800/tcp
Step 17
Verify that Barrier is running and the client can connect to the server. You can check the status and logs within the Barrier application.
Step 18
If you encounter any issues, refer to the Barrier documentation or the community forums for troubleshooting tips.
Step 19
For advanced configurations, such as setting up encryption, refer to the Barrier documentation for detailed instructions.
Step 20
To uninstall Barrier, you can use the following command:
sudo apt-get remove barrier
Step 21
Additionally, you can remove any configuration files by running:
sudo apt-get purge barrier
Step 22
Clean up any unnecessary packages by running:
sudo apt-get autoremove
Step 23
After removing Barrier, you can verify that it is no longer installed by running:
barrier --version
If the command is not found, Barrier has been successfully removed.
Step 24
For further assistance, visit the Barrier GitHub repository or the community forums.
https://github.com/debauchee/barrier
